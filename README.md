# pdf-translater

## 仮想環境の作成

仮想環境を作成する
```bash
python -m venv pdf-translater
```

仮想環境を有効化する
```bash
# 仮想環境を有効化 (Linux/macOS)
source pdf-translater/bin/activate

# 仮想環境を有効化 (Windows)
venv\Scripts\activate
```

依存関係のインストール

```bash
pip install -r requirements.txt
```

<!-- 依存関係の記録(requirements.txt を作成)
```bash
pip freeze > requirements.txt
``` -->

仮想環境の無効化
```bash
deactivate
```
